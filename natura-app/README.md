# NATURA-APP

Ecommerce platform for selling cosmetics and perfumes developed with Next.js 14, App Router, Styled Components, TanStack React Query, Zustand.

## Installation
```bash
git clone https://github.com/luizfelipe94
npm install
# dev mode
npm run dev
```
